# Typescript

Mutative support is built into the typescript types. You can use the `Draft<T>` and `Immutable<T>` types to annotate your code.

## Using TypeScript

- `castDraft()`
- `castImmutable()`
- `castMutable()`
- `Draft<T>`
- `Immutable<T>`
- `Patches`
- `Patch`
- `Options<O, F>`
